Recently Used Assets Version 1.0
================================

Usage
-----
Access the editor window under Window > Asset Management > Recently Used Assets.

Assets are added to the list whenever the editor decides. This happens whenever you 
save changes to an asset. This also happen if you create, rename or import assets.

You can also drag and drop assets as well as rightclick on them as you are used to
doing in the project window.

More recently modified assets will jump to the top of the list. Favourite an asset 
to keep it pinned to the top of the list.

Current Limitations
-------------------
You cannot rename or drag and drop assets from the Recently Used Assets Window.
